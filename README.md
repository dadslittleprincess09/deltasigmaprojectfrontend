# DeltaSigma Shop - Frontend

## Installation & Run

1. **Clone the repository**

git clone https://github.com/dadslittleprincess09/deltasigmaprojectfrontend.git
cd deltasigmaprojectfrontend

2.**Install dependencies**
npm install

3.**Create a .env file in the root:**
VITE_BACKEND_GET=https://deltasigma-api.onrender.com/api/items
VITE_BACKEND_POST=https://deltasigma-api.onrender.com/api/items

4.**Start the development server**
npm run dev (Open http://localhost:5173 in your browser.)
